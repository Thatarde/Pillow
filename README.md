# Pillow
Atualmente com o avanço da tecnologia, a utilização das IAs de forma errônea, postergando o pensamento individual e a popularização dos vídeos cada vez mais curtos, as crianças estão tendendo a exercitar pouco o cérebro com atividades que promovem o desenvolvimento cognitivo,visto que, essas atividades estimulam a atenção, concentração e percepção visual, ao mesmo tempo que melhoram a memória, a capacidade de resolução de problemas e o raciocínio lógico.


Visto isso, o projeto produzido em Python, juntamente com a biblioteca Pillow, vem com o propósito de estimular a prática destas atividades de uma forma divertida e intuitiva, gerando uma vontade de se divertir de uma maneira diferente, benéfica e próximo a tecnologia.
